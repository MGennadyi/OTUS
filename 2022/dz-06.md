# Кластер Patroni on-premise.
##### 0. Установка postgresql-14.
##### 1. Установка ETCD:
```
apt install etcd -y
rm -rf /var/lib/etcd/*
mkdir /var/lib/etcd
ls -l /var/lib/
sudo chown -R etcd /var/lib/etcd
vim /etc/default/etcd
systemctl stop etcd
vim /etc/default/etcd
```
